# Bounty Fresh Website Demo

This package has:
- Bounty Fresh-inspired content and red/yellow visual theme
- Product filters that work: All products / Whole chicken / Chicken cuts / Ready-to-cook
- Local SVG product illustrations in `assets/images/`
- HTML, CSS, and JavaScript files ready for GitHub Pages

## Before publishing
Replace the demo contact details, store links, product descriptions, and other brand information with approved Bounty Fresh content.

## Upload to GitHub Pages
Upload the contents of this folder to the root of your repository:
```
index.html
styles.css
script.js
assets/
```
Then refresh your GitHub Pages website with Ctrl + F5.
